UNT Javascript Workshop

Javascript Workshop Length: 3.5 hours A gentle but fast paced introduction to Javascript from a web development perspective. This workshop requires no web server, everything is done locally. You will need a code editor. Any will do. It is very helpful to have the directory structure displayed. You will need to bring a laptop (pc or mac) with a functioning web browser (preferably firefox), an editor for editing code. Visual Studio Code is pretty good (this runs on a mac also!), as are a few others such as Notepad++, Sublime, Textmate etc.,

Fast Intro to Javascript and jQuery

  • What is Javascript
  • What is jQuery
  • Assignments, Loops and Arrays
  • Control Statements
  • Objects
  • Function Prototypes
  • JSON and Javascript
  • Debugging javascript
  • jQuery and the DOM (HTML)
  • jQuery and CSS

    • What will we be building

    We will be building a Patient Appointments Web Application that gives doctor's offices the ability to schedule making appointments for their patients to see one of their physicians.

    For this workshop, all storage is done via local storage. All you need is a decent code editor and a browser to be able to follow the workshop.

    Server side handling of information is a topic for another more in depth look at web development.

    If you want add more capability to this, here are some ideas

  • Calendar view of upcoming appointments
  • Detecting overlapping appointments
  • Adding multiple doctors to the practice
  • Adding multiple appointment schedulers to the practice (needs server side database)

    • Before starting, test your browser
